Difference Between i++ and ++i

We can determine the differences base on the order of evaluation during the increment operation

  • In i++ which is the post-increment, the current value of i is used first, and then i is incremented.
  • Whereas in ++i which is the pre-increment, the value of i is in cremented first, and then the updated value is used.
